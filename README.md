# Proyek2_Data-Analyst-Project-Business-Decision-Research

Proyek ini membahas mengenai penurunan pelanggan dalam membeli di website toko atau customer churn. Selain itu, juga akan membuat model klasifikasi dengan Logistic Regression. Untuk penjelasan logistic regression dapat dilihat [disini](https://towardsdatascience.com/logistic-regression-using-python-sklearn-numpy-mnist-handwriting-recognition-matplotlib-a6b31e2b166a)

Proyek ini terbagi menjadi tiga bagian:
1. Data Preparation
- Importing data: Melakukan import data_retail.csv ke python environment.
- Cleansing data: Melakukan pembersihan dan modifikasi data sehingga siap digunakan untuk analisis lebih lanjut.
2. Data Visualization 
- Mendapatkan insight dari hasil visualisasi yang telah dibuat, dari visualisasi ini juga dapat melihat pola dari customer churn
3. Basic Stats Model
- Mendapatkan insight dari pembuatan model klasifikasi Logistic Regression
